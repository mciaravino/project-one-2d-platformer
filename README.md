# Project One - The 2D Platformer

A complete 2D platformer built in Godot 4. Play through multiple levels, 
avoid enemies, and reach the end to trigger the win cutscene.

🎮 **[Play it on itch.io](https://markcia.itch.io/project-one)**

## About
Project One was developed as part of a game design course. It features 
custom pixel art, sound effects, multiple levels, and a full UI built 
entirely in GDScript.

## Features
- Multiple platformer levels
- Custom tilesets and pixel art
- Enemy actors and player movement
- Sound effects
- Win cutscene

## How to Run Locally
1. Download and install [Godot 4](https://godotengine.org/)
2. Clone this repository
3. Open `project.godot` in Godot 4
4. Press Play
